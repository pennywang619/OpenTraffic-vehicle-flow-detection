4. `06_lane_queue_length_detection_sample`
5. `07_overflow_detection_sample`
6. `08_lane_line_detection_sample`

## Bundle Layout

Each submodule bundle contains:

1. `.so` in `artifacts/`
2. result examples in `examples/`
3. public manifest in `manifests/public_manifest.json`
4. validation summary in `validation/public_validation_summary.json`
5. checksum file in `checksums.txt`
